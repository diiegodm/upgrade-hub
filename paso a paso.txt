git hub 

comandos:

git clone (url): nos ayuda a clonar el repositorio 

git Branch -b (nombre de la nueva rama) : sirve para crear una nueva rama 

gti status : (nos sirve para chekar sobre los estados o movimientos que se han tenido) 

git add . (nos sirve para subir o agregar todos los archivos dentro del repositorio)
git add (nombre del archivo) : solo nos agrega el archivo seleccionado

git commit -m : nos ayuda a crear un comentario para nuestro repositorio 

git push : con este lanzamos o subimos nuestros repositorios 

git pull: nos ayuda a descargar los repositorios o actualizaciones que han hecho 


git marge : nos ayuda a unir las ramos y juntar los trabjamos o actualizaciones hechos ( de los demas compañeros )

git branch -D (nombre de la rama) : nos ayuda a eliminar la rama especificada

git checkout (nombre de la rama ): nos aydua a cambiar de rama
